# Flutter Anime API Demo

Este proyecto es una aplicación móvil construida con Flutter que consume la API de [Jikan](https://jikan.moe/) para mostrar una lista de animes. Los datos incluyen el título del anime y la imagen correspondiente.

## Características

- Listado de animes con sus imágenes.
- Refrescado de la lista de animes mediante un botón.
- Diseño en formato de cartelera de cine.

## Requisitos

- Flutter 2.0 o superior.
- Conexión a Internet para acceder a la API de Jikan.
